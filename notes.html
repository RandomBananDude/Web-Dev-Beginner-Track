When it comes to CSS it is put in style tags nexted inside of the head tags, but instead of going to every element and
putting a style attribute for every element, we can make a style element which is going to be out rule set for every element
A CSS ruleset has two parts a selector and a declaration block, a CSS ruleset is also put between style tags
  • Selector is the name of the Elements
  • Declartion block has all the property and values you want to set or modify for the Element

To apply specific Properties and values to different instances of the same element we can use classes, and we can do that by just entering a class as an attribute and naming it whatever we like, later on after we do that, when we come to styling we can use the .class as our selector

Another way we can do this is by using id, but the difference between ids and classes is that there can be many classes
for an element but only one ID, but to apply style to elements in an id, instead of putting a dot before the selector we use the #
You can also combine two selectors by using descendant selectors by putting the parent selector first then the child

To apply two classes in one element, we simply put the class names next to each other seperated by a space
So summing up selectors we have three types type selector(entire element), class selector and ids
